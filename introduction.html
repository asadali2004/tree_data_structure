A. What is a Tree Data Structure?

A Tree is a non-linear hierarchical data structure that consists of nodes connected by edges. It simulates a hierarchical tree structure with a root value and subtrees of children with a parent node, represented as a set of linked nodes. Unlike real trees, the tree data structure is typically drawn upside down with the root at the top and the leaves at the bottom.

Key Characteristics:

  • Hierarchical: Data is organized in a parent-child relationship.
  • Non-linear: Elements are not stored sequentially.
  • Connected: All nodes are connected, forming a single connected component.
  • No Cycles: A tree is an acyclic graph, meaning there are no loops or cycles.

C. Basic Tree Terminology

To understand trees, it's crucial to be familiar with the following terms:

Node
A fundamental unit of a tree, containing data and links/references to its child nodes.
Root
The topmost node in a tree. It has no parent. Every tree has exactly one root node.
Edge
The link or connection between two nodes.
Parent
A node that has one or more child nodes.
Child
A node that has a parent node.
Sibling
Nodes that share the same parent.
Leaf (External Node)
A node that has no children. These are the nodes at the bottom of the tree.
Internal Node
A node that has at least one child. (All nodes except leaf nodes).
Path
A sequence of connected nodes from one node to another. The length of a path is the number of edges on the path.
Depth of a Node
The length of the path from the root to that node. The root node has a depth of 0.
Height of a Node
The length of the longest path from that node to a leaf node in its subtree. The height of a leaf node is 0.
Height of a Tree
The height of its root node. Alternatively, it's the maximum depth of any node in the tree.
Level of a Node
The depth of the node plus one. If root is at depth 0, it's level 1.
Subtree
A tree formed by a node and all of its descendants. Any node in a tree can be considered the root of a subtree.
Degree of a Node
The number of children a node has.
Forest
A collection of disjoint trees.
